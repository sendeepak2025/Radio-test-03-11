## ğŸ› Known Issues & Solutions

### Issue 1: Orthanc Not Starting
**Symptom:** Service fails to start
**Cause:** Port 8042 already in use
**Solution:**
```cmd
netstat -ano | findstr :8042
taskkill /PID <PID> /F
net start orthanc
```

### Issue 2: Firewall Blocking
**Symptom:** Port tests fail
**Cause:** Windows Firewall rules
**Solution:**
```cmd
netsh advfirewall firewall add rule name="DICOM" dir=in action=allow protocol=TCP localport=4242
```

### Issue 3: CORS Errors (Web App)
**Symptom:** API calls fail in browser
**Cause:** Orthanc CORS not configured
**Solution:** Add to orthanc.json:
```json
"RemoteAccessAllowed": true,
"HttpsCACertificates": ""
```

### Issue 4: Network Timeout
**Symptom:** Tests take too long
**Cause:** Slow network or firewall
**Solution:** Increase timeout in code or check network
